# Narration audio

Human-recorded lesson narrations, produced with the Voiceover Studio (`../studio.html`).

- File naming: `{course-id}-{lessonIndex}.{m4a|mp3|webm}` — e.g. `growth-mindset-0.m4a`.
  The studio generates the correct name automatically; do not rename.
- The app checks for these files per lesson and plays them instead of the automatic voice.
- Lessons without a file fall back to browser text-to-speech automatically.
